Architecture

How It Works

An 8-stage AI pipeline from raw content to validated quiz questions.

01
📄

Content Ingestion

Parse PDFs, text files, or raw content using PyMuPDF with intelligent structure detection.

02
✂️

Semantic Chunking

Split text into overlapping chunks respecting section boundaries and concept coherence.

03
🔑

Concept Extraction

Extract keyphrases and entities using KeyBERT + spaCy NER for targeted question focus.

04
🗄️

FAISS Indexing

Build a vector database of chunks for retrieval-augmented generation (RAG).

05
🤖

LLM Generation

Generate questions using FLAN-T5 with Bloom's Taxonomy-guided prompting for difficulty control.

06
🎯

Distractor Gen

Create plausible wrong options using LLM + semantic similarity for high-quality MCQs.

07
✅

NLI Validation

Verify factual grounding with DeBERTa NLI, grammar checking, and structural validation.

08
📊

Output Formatting

Structure validated questions as JSON with metadata, explanations, and source context.
